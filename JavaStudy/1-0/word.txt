■ SQL
SQL
 データベース言語
 データベースを管理するソフトウェアを操作・制御することが目的
DB2
 IBM社が提供するリレーショナル型データベース管理システム(RDBMS)
MySQL
 Oracleが開発してサポートするオープンソースのSQLリレーショナルデータベース管理システム
 データを保存したり管理したりするシステム
Maria DB
 MySQLの派生版
 幅広いOSを選択できる、クラスターに対応しており、大規模な処理を行える
Oracle DB
 Oracle社が開発・リリースしているデータベース(有償)
 複雑な処理を実行でき、サポート体制も充実してる
PostgreSQL
 リレーショナルデータベース(RDB)の作成や操作、管理ができるオープンソースのデータベース管理システム(DBMS)
 拡張性が高い
SQLite
 オープンソースで軽量のRDBMS（軽量・コンパクト・省メモリ）
 設定不要でライブラリとして使用できる
SQL Server
 Microsoft社が開発したRDBMS
 企業で利用されることが多い、Windows OSとの相性がよい
CRUD
 create(登録)、read(参照)、update(更新)、delete(削除)機能をまとめた表現 
トランザクション
 複数の処理を一つにまとめたもの
 複数の処理は分離させることはできない
コミット
 データベースに結果が反映される
 一度コミットすると、処理を取り消してデータを元に戻すことはできない
ロールバック
 実行されるとトランザクション中の処理が取り消され、トランザクション実行前の状態に戻りトランザクションが終了する
排他制御（排他ロック）
 あるトランザクションが共有資源にアクセスしている時は他のトランザクションからはアクセスできないようにして直列に処理されるように制御すること
楽観ロック
 データそのものに対してロックは行わずに、更新対象のデータがデータ取得時と同じ状態であることを確認してから更新することで、データの整合性を保証する方式
悲観ロック
 更新対象のデータを取得する際にロックを掛けることで、他のトランザクションから更新されないようにする方式
実行計画
 ユーザによって発行されたクエリを実行するための手順書
 主に、テーブルからのデータ取得方法や、テーブルの結合方法が書かれている
チューニング
 SQL文のパフォーマンスを、特定の測定可能かつ達成可能な目標に合うように向上させる反復プロセス
パフォーマンス
 性能、能力
スロークエリ
 SQLの実行時間が指定した時間よりもかかってしまったSQLを全て出力することができる
クエリログ
 MySQLサーバが実行した全てのSQLクエリを出力するログ
IDE（統合開発環境）
 テキストエディタ、コンパイラ、デバッガを1つにまとめたツール
Eclipse
 IDEの一つ
 Javaの開発環境が含まれている
NetBeans
 プロジェクト管理機能や入力支援機能を備えたオープンソースのIDE
 GUIアプリケーションを直感的に開発できる
SpringToolSuite
 Springベースのアプリケーション開発を行うために提供される開発ツール
DBツール
 データベース管理タスクに使用できるツール
A5:SQL Mk-2
 SQL文の入力支援やER図作成等の機能を備えた高機能なSQL開発環境
pgAdmin
 PostgreSQLの管理ツールの一つ
 遠隔のデータベースサーバをGUIで管理することができる
DBeaver
 マルチプラットフォーム対応のSQLクライアントツール
MySQL Workbench
 MySQLの公式サイトにてMySQL Serverと共に配布されている無料ツール
Sequel Pro
 MySQLのクライアントツール（Mac用のGUIアプリケーション）
TeamSQL
 対応DBが多く拡張機能が豊富
 SQLの実行履歴が残る
